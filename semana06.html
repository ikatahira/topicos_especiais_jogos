Atividade 2 — Protótipo de App Educacional Gamificado
+80 XP

Crie o protótipo (Figma ou similar) de um app educacional gamificado para ensinar um conceito técnico (programação, física, história) com pelo menos 3 níveis de Bloom presentes.

Passos
  1. Escolha o conteúdo a ensinar e o público-alvo (faixa etária, nível)
  2. Projete a progressão: como o aluno avança pelos 6 níveis de Bloom?
  3. Crie o sistema de recompensas: XP, badges, progressão visual
  4. Prototipe as telas principais: dashboard, missão, feedback, conquista
  5. Teste com 2-3 colegas e documente os feedbacks recebidos
Entrega

Protótipo Figma (mínimo 8 telas) + documento de design explicando cada decisão pedagógica + relatório de testes

🧮
Atividade 3 — Algoritmo de Adaptive Learning Simplificado
+70 XP

Implemente uma versão simplificada de adaptive learning: sistema que ajusta dificuldade das próximas perguntas baseado no desempenho das anteriores.

Passos
  1. Defina 3 níveis de dificuldade para um conjunto de 15 perguntas (5 fáceis, 5 médias, 5 difíceis)
  2. Implemente a lógica: se acerta 2 seguidas → sobe nível; se erra 2 → desce nível
  3. Crie interface simples (web ou Unity) que mostra a pergunta adequada ao nível atual
  4. Adicione feedback visual que mostre ao usuário onde está na curva de dificuldade
  5. Teste com 5 pessoas e meça: as pessoas que usaram adaptive learning X linear aprenderam melhor?
Entrega

Código funcional + documento explicando o algoritmo + comparativo de resultados dos testes (adaptive vs linear)
