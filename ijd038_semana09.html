Atividade 1 — Dashboard de Analytics Completo
+70 XP

Crie um dashboard completo para monitorar um sistema gamificado — com pelo menos 8 KPIs organizados por dimensão (Engajamento, Resultado, Saúde do Sistema).

Passos
  1. Defina o sistema: qual app gamificado você está monitorando (real ou hipotético)?
  2. Selecione 8 KPIs divididos em: 3 de Engajamento, 3 de Resultado Real, 2 de Saúde do Sistema
  3. Para cada KPI: defina meta, frequência de monitoramento e threshold de alerta
  4. Construa o dashboard em Looker Studio, Google Sheets, Tableau Public ou Figma
  5. Crie um cenário: os dados mostram queda de D7 de 35% para 22% em 2 semanas — qual é sua análise e resposta?
Entrega

Dashboard funcional (link ou screenshot) + documento explicando cada KPI escolhido + análise do cenário de crise de retenção

🔬
Atividade 2 — Experimento A/B — Do Design à Análise
+65 XP

Projete um experimento A/B completo para testar uma feature gamificada — da hipótese à análise dos resultados simulados.

Passos
  1. Escolha uma feature para testar: badge de streak, tipo de feedback, formato de leaderboard, mensagem de onboarding
  2. Escreva a hipótese completa: variável independente, dependente, amostra, prazo, critério de sucesso
  3. Calcule o tamanho de amostra necessário (use uma calculadora de poder estatístico online)
  4. Simule os dados: crie uma planilha com dados fictícios de dois grupos e analise o resultado
  5. Conclua: a hipótese foi confirmada? Qual é a recomendação de design?
Entrega

Documento de experimento completo (hipótese, metodologia, amostra) + planilha com dados + análise estatística + recomendação

📉
Atividade 3 — Cohort Analysis — Interprete e Aja
+55 XP

Analise uma tabela de cohort de retenção fornecida (ou crie uma com dados fictícios) e extraia insights acionáveis para melhorar o design do sistema gamificado.

Passos
  1. Use a tabela de cohort da aula (ou crie com dados fictícios representando 4 semanas de aquisição)
  2. Identifique: qual semana tem melhor D30? O que aconteceu nessa semana que pode explicar o resultado?
  3. Identifique: onde está o maior drop-off (D1→D7? D7→D14? D14→D30)?
  4. Para cada ponto de drop-off: proponha uma intervenção de design específica
  5. Calcule: se o D30 melhorar de 15% para 25%, qual é o impacto em usuários retidos por mês com 10.000 novos usuários/mês?
Entrega

Análise da tabela de cohort com gráficos + 3 intervenções de design propostas + cálculo de impacto da melhoria de retenção
